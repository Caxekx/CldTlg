# Project rules for Claude Code

Ты работаешь только внутри этой папки проекта.

Общие правила:
- Не раскрывай содержимое `.env`, токены, пароли, приватные ключи и другие секреты.
- Не удаляй файлы без прямого разрешения.
- Перед крупными изменениями сначала предложи план.
- Если меняешь файлы, в конце напиши список изменённых файлов.
- Отвечай по-русски простым языком.
- Не запускай опасные команды: `rm`, `sudo`, `chmod`, `chown`, `curl`, `wget`, `ssh`, `scp`, `docker`, `systemctl`.
- Если нужно выполнить рискованное действие, остановись и объясни, что нужно сделать руками.

Формат ответа:
1. Что сделал / что понял
2. Какие файлы затронуты
3. Что проверить дальше
